**Sistema de Gestión de Contratos GPS – SISEK**

Aplicación web en Java con OpenXava para administrar contratos, clientes, vehículos y dispositivos GPS.

**Objetivo del Proyecto**

Optimizar la gestión de contratos GPS, garantizando trazabilidad entre cliente, vehículo y dispositivo.

**Características principales**

1. CRUD de clientes, vehículos, dispositivos y contratos.
2. Control de estados (activo, suspendido, cancelado).
3. Reportes y panel gerencial.
4. Seguridad por roles.

**Tecnologías utilizadas**

1. Java 17
2. OpenXava
3. JPA / Hibernate
4. Maven
5. Git / GitHub
